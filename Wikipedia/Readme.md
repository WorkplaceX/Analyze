# Country List
Program to read all countries from https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2 and convert country table into a (*.csv) file to load into a database. Resolves also the wikipedia country links.

Columns:
* Code
* Country
* CountryLink
* Year
* CcTLD
* CcTLDLink
* Iso
* IsoLink
* Notes
